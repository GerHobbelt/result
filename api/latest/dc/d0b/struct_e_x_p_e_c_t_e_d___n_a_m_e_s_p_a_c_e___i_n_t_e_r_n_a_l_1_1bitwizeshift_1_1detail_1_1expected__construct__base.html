Detailed Description

template<typename T, typename E>
struct EXPECTED_NAMESPACE_INTERNAL::bitwizeshift::detail::expected_construct_base< T, E >

Base class of assignment to enable construction and assignment.

This class is used with several pieces of construction to ensure trivial constructibility and assignability:

  • expected_trivial_copy_ctor_base
  • expected_trivial_move_ctor_base
  • expected_copy_assign_base
  • expected_move_assign_base
Template Parameters
Tthe value type
Ethe error type
